Welcome to our Chat App! This real-time messaging application is built with React Native, JavaScript, Socket.IO, Node.js, Redux, and integrates with external APIs.

## Features

- Real-time messaging: Experience instant messaging with friends and family.
- User Authentication: Secure login and registration system.
- Redux Integration: State management for smooth data flow.
- API Integration: Connects with external APIs for enhanced functionality.
- Customizable User Interface: Tailored interface for an intuitive user experience.

## Technologies Used
- ![Socket.IO](https://img.shields.io/badge/socket.io-%23000000.svg?style=for-the-badge&logo=socket.io&logoColor=white): Real-time bi-directional communication between web clients and servers. Used for feature chat with a person or group.
- ![AWS](https://img.shields.io/badge/AWS-%23232F3E.svg?style=for-the-badge&logo=amazon-aws&logoColor=white): Deploy backend with EC2 and use AWS SES for authentication with email.

- ![NPM](https://img.shields.io/badge/NPM-222222?logo=npm): Package manager for JavaScript.
- ![Expo](https://img.shields.io/badge/Expo-222222?logo=expo): Development tool for building mobile applications with React Native.

## Installation
1. Clone the repository:
    ```
    git clone https://github.com/VuLanTuong/chatty-mobile-app
    ```

2. Install dependencies for frontend and backend:

    ```
    cd chatty-mobile-app
    npm install
    ```
3. Start the frontend application:

    ```
    npm run web
    ```

4. Open your browser and navigate to `http://localhost:19006` or use the Expo app to scan the QR code and access the Chat App.
